# <a name="version-element"></a>Элемент Version

Указывает версию надстройки Office.

**Тип надстройки:** контентные и почтовые надстройки, надстройки области задач.

## <a name="syntax"></a>Синтаксис

```XML
<Version>n .n .n .n</Version>
```
