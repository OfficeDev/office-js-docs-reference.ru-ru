# <a name="formsettings-element"></a>Элемент FormSettings

Указывает исходное расположение и параметры элемента управления для почтовой надстройки.

**Тип надстройки:** почтовые.

## <a name="syntax"></a>Синтаксис

```XML
<FormSettings>
   ...
</FormSettings>
```
