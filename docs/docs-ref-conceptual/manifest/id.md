# <a name="id-element"></a>Элемент Id

Указывает уникальный идентификатор надстройки Office как GUID.

**Тип надстройки:** контентные и почтовые надстройки, надстройки области задач.

## <a name="syntax"></a>Синтаксис

```XML
<Id>string</Id>
```
